SUMO (Simulation of Urban Mobility) simulator C++ Library.
==========================================================

```bash
git clone http://github.com/rosmod/lib-sumo
cd lib-sumo
```
Then build the library.

Build Library:
-------------
If you have `catkin_make`:

```
catkin_make
```

If you have `cmake`:

```bash
cd src/sumo
cmake .
make
```
